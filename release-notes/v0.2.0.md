## 更新内容

- 新增桌面歌词窗口，可从播放器底部的桌面歌词按钮打开独立透明歌词窗口。
- 桌面歌词支持两行歌词显示，当前歌词高亮，下一句歌词提前预览；歌词缺失或纯音乐时使用单行居中提示。
- 桌面歌词支持横向和纵向显示，切换方向时会同步交换窗口宽高，并记住上次方向设置。
- 桌面歌词新增悬停控制栏，支持播放模式切换、上一首、播放/暂停、下一首和方向切换。
- 优化桌面歌词显示效果，歌词使用白色文字和黑色描边，高亮歌词继续使用主题高亮色并保留描边，提升浅色背景下的可读性。
- 新增桌面歌词双击锁定能力，锁定后隐藏背景和按钮栏，并启用鼠标点击穿透。
- 新增 `Alt + \\` 全局快捷键，用于只向桌面歌词窗口发送解锁事件，恢复鼠标交互。
- 调整 `Ctrl + \\` 全局快捷键逻辑，由隐藏/显示主窗口改为最小化/显示主窗口。
- 优化桌面歌词窗口创建逻辑，改为前端直接创建和关闭子窗口，减少后端窗口控制命令。
- 新增桌面歌词专用歌词解析工具，为后续逐行、逐字歌词渲染扩展打基础。
