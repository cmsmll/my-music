## 更新内容

- 设置页“关于”中新增 GitHub 仓库地址和当前版本信息。
- 修复设置页输入框、按钮等控件在 hover / active / focus 状态下出现异常黑边的问题。
- 新增样式配置中的歌词色，歌词高亮颜色可以单独调整。
- 默认高亮色和歌词色调整为 `#3bce82`。
- 启动窗口新增默认宽高配置，并和最小宽高保持一致，减少打包后启动尺寸异常的情况。
- 解码器命令改为异步执行，避免重型解码任务阻塞主线程。
- 曲库扫描、重载曲库、启动状态加载、歌词使用等耗时命令迁移到阻塞任务线程中执行，降低界面卡顿风险。
- 新增 `.gitattributes`，统一仓库文本文件换行为 LF，减少跨平台提交噪声。
- GitHub Release 发布脚本支持自动读取 `release-notes` 目录下的当前版本更新说明；如果同版本 Release 已存在，会同步更新说明并覆盖上传安装包。
