## 更新内容

- 新增 `Ctrl + \` 全局快捷键，用于快速显示或隐藏主窗口；窗口隐藏后再次触发会自动显示、取消最小化并聚焦。
- 优化解码结果统计，已扫描文件现在会统计所有符合“处理格式”的文件，包括成功、失败和跳过的文件。
- 解码弹窗文案从“处理”调整为“解码”，统计顺序调整为“已扫描、解码、失败、跳过”，结果展示更符合实际含义。
- 升级并适配 `symphonia 0.6.0`，更新音频时长读取逻辑以兼容新版探测 API 和轨道时长结构。
- 优化 Rust 后端依赖 features，关闭 `reqwest`、`chrono`、`webp` 的非必要默认功能，减少依赖体积和编译负担。
- 更新 `md5`、`memchr` 等后端依赖版本，并同步 `Cargo.lock`。
- 保留现有曲库格式支持和封面处理逻辑，依赖优化不改变曲库扫描、歌词搜索和音频解码的功能范围。
