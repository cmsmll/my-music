## 更新内容

- 播放核心切换为前端 `AudioEngine.vue` 封装的 `<audio>` 标签，移除旧的前端 `AudioPlayer` 封装，减少后端音频设备占用和输出设备切换问题。
- 播放进度缓存改为前端本地轻量存储，播放进度按音频事件更新，关闭应用前会刷新当前播放记录。
- 优化播放进度拖拽体验，拖拽期间前端进度条跟手更新，松开后再同步实际音频位置。
- 语义化播放模式：`shuffle` 表示随机列表，切换时会打乱当前播放列表并更新播放队列；`random` 表示随机播放，只在下一首时临时随机挑歌，不改变播放队列。
- 播放模式按钮文案调整，“随机列表”用于打乱后的播放队列，“随机播放”用于临时随机下一首。
- 歌曲详情弹窗新增文件路径、封面缓存和歌词缓存信息，并提供打开文件夹入口。
- 新增 Rust 后端打开目录命令，打开失败会写入应用日志，并把错误信息返回前端通过全局消息弹窗提示。
- 新增单例启动，重复打开应用时会唤起已有主窗口、取消最小化并聚焦，不再创建第二个实例。
- 移除未使用的频谱缓存配置，前后端配置、设置页和 README 均不再展示 `spectrum_cache_dir`。
- 更新 README，补充最近更新、播放模式说明、当前前端音频播放实现、缓存配置和后端目录结构。
